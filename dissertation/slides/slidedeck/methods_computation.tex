        \begin{itemize}

        \item Roe Solver used for fluxes
        \end{itemize}
      \item Acoustic waves prescribed in the domain.
      \item Grid stretching reduces reflections.
      \item Grid size: $\ell \times 80\ell \, (L_x \times L_y)$
